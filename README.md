# selenium base

selenium driverless browser automation base with mobile style spoofing and custom fingerprint scripts

## what it supports
1. iphone device spoofing
2. randomized mobile user agents
3. mobile screen metrics and scale factor spoofing
4. touch support and max touch point spoofing
5. hardware concurrency spoofing
6. language and dark mode spoofing
7. navigator and platform spoofing
8. canvas fingerprint spoofing
9. audio fingerprint spoofing
10. gpu and webgl spoofing
11. stealth script preloading
12. custom javascript injection on new documents
13. cache and profile directory support
14. proxy support

## notice

this is a starter base meant to be extended for your own automation workflow
webgl javascript spoofing scripts may be outdated and should be reviewed before real use
